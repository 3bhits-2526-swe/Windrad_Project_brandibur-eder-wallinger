Teammitglieder: Bogdan Brandibur - Tobias Wallinger - Luca Eder

  Das Unity Projekt soll mehrere 3D-Windraäder anzeigen, welche sich drehen.
  Die Geschwindigkeit mit der sie sich drehen hängt zusammen mit der Windgeschwindigkeit,
  welche simuliert wird. Die Windgeschwindigkeit basiert auf echten Wetterdaten aus
  verschiedensten Städten der Welt. Die Wetterdaten werden aus einer Wetter-API beschaffen
  und die Stadt kann man per Mausklick aus einer Liste an selektierten Städten auswählen.
